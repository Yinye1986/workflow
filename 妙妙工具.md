# term和shell

```shell
tty             # 显示当前终端设备
stty -a         # 显示当前终端设备的信息
bind -l         # 列出函数名称
bind -P         # 列出当前模式下的函数名称和绑定
bind -p         # 与前者不同的是这里的输出格式即 .inputrc 中的配置格式
bind -P/p -m vi # 指定模式的输出, 可选项参考 bind --help 中列出的
```

# 内建命令type

可以识别:

- 内建命令: type是一个内建命令
- 外部命令: sway是/usr/bin/sway
- 函数    : 函数定义
- 别名    : 别名定义
- sh脚本  : xxx.sh是xxx.sh
- ...

# 内建命令command

command ls 执行 ls 程序, 即使存在 ls 别名

command -v ls 返回路径或定义

# util-linux提供whereis

在内建的一系列指定的文件夹里即时搜索, 会输出:

1. 可执行文件位置
2. 源码文件位置(/usr/src)

Binary (可执行文件)：命令的主体程序，通常位于 $PATH 中的目录。

Source (源码文件)：命令的源代码文件（如果已安装）。

Manual (手册页)：命令的帮助文档，即 man 手册文件。
