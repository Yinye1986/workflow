# 安装

Google Store 版本较为落后, 从github上获取最新版本安装, v8a架构

# 配置ssh

```shell
pkg install ssh
sshd # 开启 ssh 守护进程
```

## 连接

- 默认主机名: u0_aXXX
- 默认端口  : 8022

```shell
ssh u0_aXXX@192.168.123.183 -p 8022 # PC 连接 PAD
ssh shins0u@192.168.123.48          # PAD 连接 PC
```

# Git

克隆指定设备的文件夹(需配置 ssh)

```shell
git clone shins0u@192.168.123.48:~/AAA/yazi
```

# 软链接

```shell
ln -sf /sdcard ~/sdcard
```
