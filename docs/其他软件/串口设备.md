# stty

查看串口设备配置

```shell
stty -F /dev/ttyUSB0 -a
```

# 虚拟端口

可以在本地创建虚拟端口用于调试

```shell
socat -d -d \
  pty,raw,echo=0,link=/tmp/VCOM1,mode=666 \
  pty,raw,echo=0,link=/tmp/VCOM2,mode=666
```

# 连接串口设备

## minicom

使用 minicom 连接

```shell
minicom -D /dev/ttyUSB0 -b 115200 -o -w
```
-o 不发送 Modem 初始化命令, 不创建锁文件
-w 自动换行(防止截断过长行, 或需要翻页)

## screen

使用 screen 连接

```shell
screen /dev/ttyUSB0 115200
```
