# x0.at

从标准输入中读取并使用 netcat/curl/ssh 上传

不限制文件类型

大小 512M 以内

```shell
echo "hello" | nc x0.at 9999
echo "hello" | curl -F "file=@-;filename=.txt" https://x0.at/
echo "hello" | ssh x0.at
```
# 0x0.st

从本地上传

从 url 上传

```shell
curl -F'file=@yourfile.png' http://0x0.st
curl -F'url=http://example.com/image.jpg' http://0x0.st
```

# clbin.com

从标准输入中读取并上传

使用 alias 简化命令

```shell
echo "test" | curl -F 'clbin=<-' https://clbin.com
echo "test" | clbin
```

查看时可在末尾加上 ?hl 以显示行号和高亮
