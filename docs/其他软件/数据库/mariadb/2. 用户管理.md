
# 创建用户

```sql
CREATE USER '用户名'@'主机名' [OPTS];
CREATE USER 'shins0u'@'localhost' IDENTIFIED BY 'password';
CREATE USER 'shins0u'@'%' IDENTIFIDE BY VIA unix_socket OR;
```

# 删除用户

```sql
DROP USER shins0u@'%';
```

# 修改用户密码

```sql
ALTER USER root@localhost IDENTIFIED BY '114514';
alter user "cz_hung"@'%' IDENTIFIED BY '114514';
SET PASSWORD for root@localhost = PASSWORK('mariadb')
```


# 重载权限配置

```sql
FLUSH PRIVILEGES;
```

这些情况不需要手动重载:

- CREATE USER
- DROP USER
- ALTER USER
- GRANT
- REVOKE

这是合法的修改方式

-- 直接操作表中字段
