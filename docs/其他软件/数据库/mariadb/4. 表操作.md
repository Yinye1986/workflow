# 查看当前数据库的表

```sql
SHOW TABLES;
```

# 创建表

```sql
create table 表名(
   列名 数据类型 约束,
   列名 数据类型 约束
) 表级属性;
```

常见数据类型:

数值:
- INT
- FLOAT
- DECIMAL
文本:
- VARCHAR(n)
- TEXT
时间:
- DATE
- DATETIME

## 示例

```sql
CREATE TABLE tb_name (
   id INT PRIMARY KEY AUTO_INCREMENT,
   name VARCHAR(128) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci,
   age INT)
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;
```

## 增加字段

```sql
ALTER TABLE tb_name ADD COLUMN new_column1 INT;
ALTER TABLE tb_name ADD COLUMN new_column2 VARCHAR(128) CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 修改字段数据类型

```sql
ALTER TABLE tb_name MODIFY COLUMN new_column1 VARCHAR(128);
```
