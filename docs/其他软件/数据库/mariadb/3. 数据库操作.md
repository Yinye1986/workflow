# 查看数据库

```sql
SHOW DATABASES;
```

# 创建数据库

```sql
-- CREATE DATABASE 数据库名 数据库选项;
CREATE DATABASE db_name;
CREATE DATABASE db_name IF NOT EXISTS;
CREATE DATABASE db_name IF NOT EXISTS CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

# 指定数据库

```sql
USE db_name;
```

## 关于字符集和排序规则

字符集,排序规则使用系统全局配置, 若没配置会使用默认值

表自动继承数据库设置; 也可单独指定(也就是说表和数据库的配置可以不同)

也可作为列的约束单独为每一列设定

字符型数据需要配置这两个


查看服务器默认配置

```sql
SHOW VARIABLES LIKE 'character_set_server';
SHOW VARIABLES LIKE 'collation_server';
```

# 删除数据库

```sql
DROP DATABASE db_name;
```
