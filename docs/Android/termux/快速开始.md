# 安装

Google Store 版本较为落后, 从github上获取最新版本安装

# 配置ssh

```shell
pkg install openssh
sshd
```

1. 安装 openssh
2. 执行 sshd, 启动守护进程
3. 执行 pkill -15 sshd, 退出 sshd

## 连接

```shell
ssh u0_aXXX@192.168.123.183 -p 8022 # PC 连接 PAD
```

- 默认用户名: u0_aXXX
- 默认端口  : 8022
