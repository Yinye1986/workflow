# 准备工作

1. 初始化全局配置文件
2. 添加三方板子源
3. 更新索引
4. 搜索开发板
5. 安装开发板 sdk

```shell
arduino-cli config init
arduino-cli config add board_manager.additional_urls https://arduino.esp8266.com/stable/package_esp8266com_index.json
arduino-cli core update-index
arduino-cli core search esp8266
arduino-cli core install esp8266:esp8266
```

# 创建项目

```shell
arduino-cli sketch new AAATEST
# 键入测试代码
# ...
# 绑定当前项目到 /dev/ttyUSB0 并指定开发板类型
arduino-cli board attach -p /dev/ttyUSB0 -b esp8266:esp8266:nodemcuv2
# 编译并烧录
arduino-cli compile -u
```

# 获取开发板信息

```shell
# 需要获取权限
esptool flash_id
```
# 一段测试代码

```cpp
void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, LOW);  // 点亮 (ESP8266 逻辑反转)
  delay(500);
  digitalWrite(LED_BUILTIN, HIGH); // 熄灭
  delay(500);
}
```
