### Ventoy2Disk


官方提供 GTK/QT 图形化界面, 但 WebUI 轻便好用

软件目录中执行以下命令以启用 WebUI Server

```shell
sudo bash VentoyWeb.sh
```

访问 [WebUI](http://127.0.0.1:24680) 即可使用
